# Data Notes

## Source
This project uses a simulated dataset representing budgeted and actual expenses across departments, regions, categories, and monthly periods.

## Structure
Typical fields include:
- Date / Month
- Region
- Department
- Expense Category
- Budget Amount
- Actual Amount

## Assumptions
- Currency is consistent across all records
- Budget and actual values are comparable within the same period/category
- Data is structured for a star-schema model with a calendar and dimension tables

## Privacy
No confidential or proprietary company data is included.
